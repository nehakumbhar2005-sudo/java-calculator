# 🧮 Java Swing Calculator

A modern calculator application built using **Java Swing** with a clean dark-themed user interface, custom rounded buttons, hover effects, memory functions, and advanced mathematical operations.

<img width="350" alt="Calculator UI" src="screenshot.png">

---

## ✨ Features

### Basic Operations
- Addition (+)
- Subtraction (-)
- Multiplication (×)
- Division (÷)

### Advanced Operations
- Square Root (√)
- Percentage (%)
- Power Function (xʸ)

### Memory Functions
- MC (Memory Clear)
- MR (Memory Recall)
- M+ (Memory Add)
- M- (Memory Subtract)

### User Interface
- Modern Dark Theme
- Rounded Buttons
- Hover Effects
- Responsive Layout
- Expression Display
- Large Calculator Screen

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/YOUR_USERNAME/Java-Calculator.git
```

### Open Project

Open the project in:

- IntelliJ IDEA
- Eclipse
- NetBeans

### Run

Compile and run:

```bash
javac Calculator.java
java Calculator
```

or simply run the project using your IDE.
